<!-- SNIPPET: valid-button-attr-order — wp:button attribute ordering for WP 6.6+ validation
     KEY RULES:
     - Root-level attrs (className, backgroundColor, textColor, borderColor) MUST come BEFORE style.
     - Font size goes in style.typography.fontSize (CSS var string), NOT as root-level fontSize slug.
       Root-level fontSize causes block validation failure on wp:button.
     - The has-{slug}-font-size class does NOT belong on the outer wp:button div;
       it goes on the <a> tag only when set via style.typography.fontSize.
     - Attribute order: className → backgroundColor → textColor → borderColor → style → …
-->
